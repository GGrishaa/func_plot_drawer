# График математического выражения в терминале
Консольная программа для построения графиков математических функций.  
Вы вводите выражение от одной переменной `x`, а программа рисует график звёздочками прямо в терминале.

## Возможности
- Арифметика: `+`, `-`, `*`, `/`, унарный минус;
- Скобки: `(`, `)`;
- Тригонометрия: `sin`, `cos`, `tan`, `ctg`;
- Логарифм: `ln` (натуральный);
- Квадратный корень: `sqrt`;
- Автоматический перевод в обратную польскую запись (ОПН);
- Вычисление в 80 точках и вывод на сетку 80×25 символов (можно настроить);

## Сборка
Сборка всего проекта осуществляется с помощью Makefile командой make.

## Использование
1) Запустите программу: ./graph
2) Введите математическое выражение строчными буквами без пробелов: ln(x+2)*sin(x)
3) После ввода программа покажет обратную польскую запись и нарисует график.

## Настройка области построения
По умолчанию:
- По горизонтали: x от 0 до 4π
- По вертикали: y от -1 до 1
Для настройки можно поменять значения XLEN и YLEN в `point.h`, а также изменить масштаб поля в значениях X и Y.
